# Python Antivirus

The Python Antivirus, bound to get rid of any virus out there, it's as simple as having Python 3 on your device. 

## Why Use This Tool?

No external dependencies, it's all built in, and runs quick. Exceptions are for computers if your harddrive is greater than 1 MB. The code is simplier than you think!

## How to run

Drag and drop the .py files onto your Python3 Launcher.
